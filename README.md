# To-Do List
Aplicação de lista de tarefas feita em **React**, com salvamento no `localStorage` e barra de progressão.

## Tecnologias
- React
- JavaScript 
- CSS

## Funcionalidades
- Adicionar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar como concluída
- Barra de progresso 
- Persistência de dados no navegador (localStorage)
